Created ASP.NET Web Application using database-first approach

Includes 3 different API's (Authentication API, Booking api and doctor/patient Api).
Swagger, JWT Authentication and Repository pattern is implemented in all three API's.
BookingProj API is the main API which includes NUNIT ,Swagger and Mock also.
Client MVC (i.e., MainAuth) includes all API views and interconnections.
AuthenticationAPI is used to authorize the user using correct ceredentials.
SQL connections are done in appsetting.json for the security purpose.
